Welcome to my calculator, CoolCalc!

This is a project that took several hours, and spanned across 2 days.

Features include:

  • Numbers and operators (obviously)
  • Clear, backspace and decimal
  • Keyboard support!!!!!
  • A little message when you do a certain division :)
  • Not a part of the calculator but there is a text file documenting the process of fixing the potential bugs that TOP mentioned, you can check it out if you want.

Sadly this doesn't work with scientific notation as of right now, but it still turned out pretty cool.
